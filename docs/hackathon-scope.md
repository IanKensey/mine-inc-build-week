# Mine Inc. Build Week Scope

The purpose of this hackathon is **not** to build the complete game. It is to prove the core gameplay loop.

## Included

- Escape pod
- Camera
- Asteroid movement
- Trajectories
- Mining
- Resource drops
- Resource collection
- Manufacture one module
- Attach module

## Excluded

- Pirates
- Research
- Trading
- Tech tree
- Campaign
- Power management
- Repair
- Shields
- Advanced automation
